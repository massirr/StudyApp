DP-750 Exam Prep App v1 goals:
- Studying for DP-750 exam.
- Tracking learning progress.
- Organizing exam preparation.
- Public deployment.
- Local-first functionality.
- Web browser only target.

**Why:** MVP to ship quickly with simple architecture, optimize developer productivity, and avoid premature optimization. AI used for development assistance only.
**How to apply:** Prioritize features directly supporting these goals. Avoid backend, auth, sync, or mobile features in v1.
